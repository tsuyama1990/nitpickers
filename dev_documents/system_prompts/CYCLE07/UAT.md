# CYCLE07 UAT

## Test Scenarios

### Scenario ID: SCENARIO-07-1
**Priority**: High
The user triggers a UAT failure and verifies that the Stateless Auditor analyzes the multi-modal artifacts, generating a precise, schema-compliant JSON Fix Plan.

### Scenario ID: SCENARIO-07-2
**Priority**: High
Tests auditor handling valid JSON.

### Scenario ID: SCENARIO-07-3
**Priority**: Medium
Tests auditor handling malformed JSON from LLM.

## Behavior Definitions
GIVEN a failed Playwright test state with a screenshot WHEN the Auditor Node executes THEN the OpenRouter API is successfully called with the Base64 image AND the node returns a properly validated Pydantic JSON Fix Plan.
